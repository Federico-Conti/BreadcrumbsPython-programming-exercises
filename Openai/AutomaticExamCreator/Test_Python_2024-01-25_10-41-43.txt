

1. What is the correct way to declare a variable in Python?
A. var = 5
B. var == 5
C. var := 5
D. var => 5

Correct Answer: A. var = 5


2. What is the output of the following code?
x = 5
print(x % 2)
A. 0
B. 1
C. 2
D. 5

Correct Answer: B. 1


3. What is the correct way to write a conditional statement in Python?
A. if x > 5:
B. if x = 5:
C. if x => 5:
D. if x == 5:

Correct Answer: D. if x == 5:


4. What is the correct way to import a module in Python?
A. import module name
B. from module import name
C. include module name
D. module name import

Correct Answer: B. from module import name
