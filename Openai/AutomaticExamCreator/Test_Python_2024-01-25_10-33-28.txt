

Q1. What is the syntax for declaring a variable in Python?
A. #variable
B. var variable
C. !variable
D. var = variable

Correct Answer: D. var = variable


Q2. What type of language is Python?
A. Interpreted
B. Compiled
C. Assembly
D. Machine

Correct Answer: A. Interpreted


Q3. What type of loop is used when a set of instructions need to be repeated until a condition is met?
A. For loop
B. While loop
C. Do-while loop
D. If-else loop

Correct Answer: B. While loop


Q4. What is the result of the following expression?
2 + 5 * 3
A. 23
B. 17
C. 11
D. 25

Correct Answer: B. 17
